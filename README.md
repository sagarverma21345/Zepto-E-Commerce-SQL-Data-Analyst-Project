# Zepto-E-Commerce-SQL-Data-Analyst-Project
Completed a real-world SQL Data Analyst portfolio project using a Zepto e-commerce inventory dataset, simulating actual retail and quick-commerce business analysis workflows. Worked on end-to-end data analysis including database creation, data cleaning, exploratory data analysis (EDA), and business insight generation using PostgreSQL.

Key Highlights:
Designed and created a structured SQL database for managing e-commerce inventory data
Imported and cleaned raw CSV data by handling UTF-8 encoding issues, null values, and invalid pricing records
Converted product prices from paise to rupees for better readability and analysis consistency
Performed Exploratory Data Analysis (EDA) to analyze product categories, stock availability, duplicate SKUs, and pricing patterns
Wrote advanced SQL queries to generate business insights related to discounts, revenue estimation, inventory analysis, and value-for-money products
Identified top-performing categories, highly discounted products, expensive out-of-stock items, and inventory weight distribution
Used aggregate functions, filtering, grouping, CASE statements, sorting, and subqueries to solve real business problems
Simulated how data analysts work in e-commerce and retail domains using real-world inventory data
Tools & Skills Used:

PostgreSQL, SQL, Data Cleaning, EDA, Business Analysis, Aggregate Functions, CASE Statements, Joins, Subqueries, Data Modeling

Project Outcome:

Built a strong hands-on understanding of how SQL is used in real-world e-commerce analytics to support pricing strategy, inventory management, and business decision-making.
